Esse é um sistema web para gestão de inventário
e análise de dados estratégicos, visando otimizar a compreensão do fluxo de produtos e a gestão
assertiva de estoques.

A arquitetura do sistema foi estruturada com o emprego
de tecnologias baseadas em JavaScript, incluindo Node.js e React para o desenvolvimento e
MySQL para o banco de dados.

Os resultados demonstram uma solução capaz de automatizar o
registro de entradas e saídas, além de fornecer métricas visuais sobre o desempenho das vendas
e níveis de estoque.
